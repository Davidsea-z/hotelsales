## 文案策略

## 本地开发指南

### 环境要求
- Node.js 18+
- npm 10+

### 安装依赖
```bash
cd /home/user/webapp
npm install
```

### 启动开发服务
```bash
# 构建项目
npm run build

# 使用PM2启动
pm2 start ecosystem.config.cjs

# 查看日志
pm2 logs webapp --nostream

# 重启服务
fuser -k 3000/tcp 2>/dev/null || true
pm2 restart webapp

# 停止服务
pm2 delete webapp
```

### 访问地址
- 本地开发: http://localhost:3000
- 沙盒环境: https://3000-i71tqx6y3m4b1v6z8xk6l-dfc00ec5.sandbox.novita.ai

1. **构建项目**


2. **部署到Cloudflare**
```bash
npx wrangler pages deploy dist --project-name microconnect-hotel
```

3. **配置环境变量（如需要）**
```bash
npx wrangler pages secret put API_KEY --project-name microconnect-hotel
```
